# React Assignment Projects

Collection of React + Vite mini-projects built for API integration, UI practice, and state management exercises.

## Projects

| Project | Folder | Description | Live Demo |
| --- | --- | --- | --- |
| Authentication App | `AuthenticationApp` | Register/login/logout flow with token-based auth and protected profile fetch. | https://authentication-app-two-topaz.vercel.app/ |
| Quotes Gallery | `Quotes` | Browse, search, and copy inspirational quotes. | https://random-quotes-omega-ten.vercel.app/ |
| Random Users | `Random Users` | Fetch and display profile cards for random users. | https://react-projects-l573.vercel.app/ |
| Random Cats | `Random_cat` | 3D-style coverflow carousel of cat cards. | https://random-cats-zeta.vercel.app/ |
| Random Products | `Random_Products` | Product discovery grid with pricing, ratings, and stock info. | https://random-products-eight.vercel.app/ |
| Random Jokes | `randomjokes` | Joke browser with featured joke and copy actions. | https://react-projects-sslh.vercel.app/ |
| Random Meal | `RandomMeal` | Meal showcase cards with category and quick CTA styling. | https://random-meal-mu.vercel.app/ |
| YouTube Listing | `youtube-listing` | Responsive video feed with metadata parsing and loading skeletons. | https://youtube-listing-lemon.vercel.app/ |

## Common Stack

- React 19
- Vite
- Tailwind utility classes (used directly in JSX)
- FreeAPI public endpoints

## Run Any Project Locally

1. Open a terminal in the project folder.
2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

4. Open the local URL shown in terminal (usually http://localhost:5173).

## Notes

- Each app is independent and has its own `package.json`.
- API responses are handled per project with app-specific UI and UX states.
