# Random Cats

Interactive cat showcase with a 3D coverflow-style carousel effect.

## Live Demo

https://random-cats-zeta.vercel.app/

## Screenshot

![Random Cats Screenshot](https://github.com/user-attachments/assets/71cc0f79-e5cc-4b03-89d5-34b8b1cde4e7)

## Features

- Fetch cat items from FreeAPI
- Render up to 9 cards in a centered coverflow layout
- Click any card to focus it as the active slide
- 3D transforms with opacity and depth transitions
- Mobile-to-desktop responsive visuals

## Tech Stack

- React
- Vite
- Fetch API
- Tailwind utility classes

## API Endpoint

`https://api.freeapi.app/api/v1/public/cats`

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - start development server
- `npm run build` - build for production
- `npm run preview` - preview production build

## Project Goal

Practice custom card transforms, carousel math, and rich interactive layouts.
